# CodePTIT Helper

CodePTIT Helper là tiện ích mở rộng (Chrome Extension) hỗ trợ học và làm bài trên nền tảng CodePTIT mới.

## Chức năng

- Sao chép toàn bộ đề bài
- Sao chép Sample Input
- Sao chép Sample Output
- Sao chép Testcase
- Tải xuống `input.txt`
- Tải xuống `output.txt`
- Sao chép mã bài
- Sao chép tên bài

## Cài đặt

### Bước 1

Tải phiên bản mới nhất tại mục **Releases**.

### Bước 2

Giải nén file zip.

### Bước 3

Mở trình duyệt Chrome.

Truy cập

```
chrome://extensions
```

### Bước 4

Bật

```
Developer mode
```

### Bước 5

Chọn

```
Load unpacked
```

### Bước 6

Chọn thư mục

```
CodePTIT-Helper
```

Hoàn tất.

## Cách sử dụng

Mở bất kỳ bài nào trên

https://code.ptit.edu.vn

Ví dụ

https://code.ptit.edu.vn/submit/ICPC0107

Extension sẽ tự động hiển thị thanh công cụ.
